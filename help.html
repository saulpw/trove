How trove works

Trove is a shared collection of links organized by tags. Each link can have one or more tags (e.g. games, retro, tools).

Browsing by tag

Click any #tag to see all links with that tag. The URL updates to reflect the filter — e.g. /games shows all links tagged "games".

Tag intersections

Combine tags by adding more path segments. For example:

  • /games — links tagged "games"
  • /games/retro — links tagged both "games" and "retro"
  • /games/retro/free — links tagged all three

The heading shows the active filters joined by ∩ (intersection symbol). Click any filter in the heading to navigate to just that tag.

Excluding tags

Prefix a tag with - to exclude it:

  • /games/-retro — links tagged "games" but not "retro"

The tag menu

Hover over any tag on a link card to see a popup menu with three options:

  • → /tag — navigate to that tag's page (replaces current filters)
  • + /current/tag — add this tag to the current intersection (narrow results)
  • − /current/-tag — exclude this tag from the current view

The tag sidebar

On wider screens, a sidebar appears on the right showing all tags present in the currently displayed links, sorted by frequency. Click a sidebar tag to see the same menu options. Tags already in the active filter are hidden from the sidebar.

Sorting & filtering

  • Sort — newest first (default), oldest first, alphabetical, or random
  • Time period — filter links by when they were added (last day, week, month, year, or all time)

Hiding links

Each link card has a "Hide" button in the top-right corner. Hidden links are stored locally in the browser and can be toggled back via the "N hidden" link in the link count.

Favorites

Each link card has a ♥ heart icon in the lower-right corner. Click it to add the link to your favorites (the heart turns red). Click again to remove it. Favorites are stored locally in the browser.

To view all your favorited links, click your username in the top-right corner and select My favorites. The favorites page displays links just like a tag page, with sorting and tag sidebar.

Adding links

Signed-in users can add links using the form at the top of the page. Tags are space-separated. A bookmarklet is available in the footer for quick submissions from any page.
